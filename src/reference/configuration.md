# Configuration

Learn how to configure Isaac for your needs.

## Configuration File

Isaac uses a configuration file located at `~/.config/isaac/config.toml`.

### Basic Configuration

```toml
[general]
option1 = "value1"
option2 = "value2"
```

### Advanced Configuration

```toml
[advanced]
feature1 = true
feature2 = false
```

## Environment Variables

Isaac supports the following environment variables:

- `ISAAC_VAR1`: Description
- `ISAAC_VAR2`: Description
